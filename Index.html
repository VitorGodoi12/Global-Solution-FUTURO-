Solução Proposta – "Futuro+"

Futuro+ é uma plataforma inteligente que ajuda jovens e trabalhadores a se prepararem para o Futuro do Trabalho.

Ela combina IA, dados, educação e tecnologia para:

  • Recomendar profissões do futuro
  • Indicar trilhas de estudo de acordo com o perfil do usuário
  • Orientar sobre habilidades necessárias para o mercado até 2030
  • Apoiar pessoas que precisam de reskilling (aprender algo novo) e upskilling (melhorar o que já sabem)
  • Democratizar o acesso à informação sobre trabalho e carreira

Conexão com os ODS da ONU

A plataforma se conecta aos ODS da ONU, especialmente:

✓ ODS 4 – Educação de Qualidade

Promovemos educação acessível e de qualidade para todos.

✓ ODS 8 – Trabalho Decente e Crescimento Econômico

Apoiamos o desenvolvimento de habilidades para trabalho digno e crescimento sustentável.

A solução será apresentada através de:

  • Um chatbot inteligente que conversa com o usuário
  • Um site responsivo explicando carreiras e estudos
  • Um sistema Java simulando lógica da plataforma
  • Um banco de dados relacional armazenando usuários, cursos e profissões
  • Um programa Python realizando análises simples
  • Documentação + backlog para explicar o funcionamento do projeto
