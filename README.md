# Futuro+ | Plataforma de Orientação Profissional

O **Futuro+** é uma plataforma inteligente desenvolvida para ajudar jovens e trabalhadores a se prepararem para o **Futuro do Trabalho**. A solução combina **Inteligência Artificial (IA), dados, educação e tecnologia** para oferecer orientação profissional e educacional.

Este repositório contém o **código-fonte do Frontend (Site Responsivo)** da plataforma.

## Funcionalidades Principais

A plataforma Futuro+ visa:

*   Recomendar **profissões do futuro**.
*   Indicar **trilhas de estudo** de acordo com o perfil do usuário.
*   Orientar sobre **habilidades necessárias** para o mercado até 2030.
*   Apoiar pessoas que precisam de **reskilling** (aprender algo novo) e **upskilling** (melhorar o que já sabem).
*   Democratizar o acesso à informação sobre trabalho e carreira.

## Conexão com os ODS da ONU

O projeto está alinhado com os Objetivos de Desenvolvimento Sustentável (ODS) da ONU, focando especialmente em:

*   **ODS 4 – Educação de Qualidade:** Promovendo educação acessível e de qualidade para todos.
*   **ODS 8 – Trabalho Decente e Crescimento Econômico:** Apoiando o desenvolvimento de habilidades para trabalho digno e crescimento sustentável.

## Tecnologias Utilizadas (Frontend)

O frontend do projeto é uma aplicação web estática, desenvolvida com as seguintes tecnologias:

| Tecnologia | Descrição |
| :--- | :--- |
| **HTML5** | Estrutura semântica de todas as páginas. |
| **CSS3** | Estilização completa, incluindo a responsividade (`global.css`, `index.css`, etc.). |
| **JavaScript** | Lógica de interação simples, como o menu de navegação (`contato.js` e outros scripts). |


## Componentes da Solução Completa

O frontend é apenas uma parte da solução completa "Futuro+", que também inclui:

*   **Chatbot Inteligente:** Para conversação e interação com o usuário.
*   **Sistema Java:** Simulação da lógica de negócios da plataforma.
*   **Banco de Dados Relacional:** Armazenamento de usuários, cursos e profissões.
*   **Programa Python:** Para análises de dados e lógica de IA.
*   **Documentação e Backlog:** Para gestão e explicação do projeto.
